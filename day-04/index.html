Practice CSS Box Model and Selectors

When to Use Which CSS Selector?

  • ID Selector (#id): Use when you want to style a unique element on the page. Each ID should be used only once per page.
  • Class Selector (.class): Use when you want to apply the same style to multiple elements. Classes can be reused on any number of elements.
  • Tag Name Selector (tagname): Use when you want to style all elements of a certain type (e.g., all <p> tags).
  • Attribute Selector ([attribute]): Use when you want to style elements that have a specific attribute, regardless of its value.
  • Attribute and Value Selector ([attribute="value"]): Use when you want to style elements with a specific attribute and a specific value.
  • Tag + Attribute Selector (tag[attribute="value"]): Use when you want to style a specific type of element with a specific attribute and value.

Summary: Use ID for one-of-a-kind elements, class for groups, tag for all of a type, and attribute selectors for more specific targeting.
Choosing the right selector keeps your CSS efficient and maintainable.

Select Element by Tag Name

p { border: 1px solid black; padding: 10px; }

Select Element by Attribute

[id] { background-color: lightgray; }

Select Element by Attribute and Value

[id="p3"] {color: red;}

Select Element by Tag Name along with Attribute and Value

p[id="p4"] {color: blue;}

Select Element by ID

#p5 {background-color: blue;}

Select Element by Class Name

.paragraph {background-color: lightseagreen;}
